# Projeto Lâmpada

Projeto simples em JavaScript para praticar manipulação do DOM e eventos.

A aplicação simula uma lâmpada que pode ser ligada, desligada e quebrada a partir de interações do usuário.

## Funcionalidades
- Ligar a lâmpada
- Desligar a lâmpada
- Ligar ao passar o mouse
- Desligar ao retirar o mouse
- Quebrar a lâmpada com duplo clique
- Bloquear ações após a lâmpada estar quebrada

## Conceitos praticados
- Manipulação do DOM
- Eventos (`click`, `mouseenter`, `mouseleave`, `dblclick`)
- Funções
- Condicionais
- Controle de estado

## Tecnologias
- HTML
- CSS
- JavaScript (Vanilla JS)
