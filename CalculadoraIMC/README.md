# Calculadora IMC

Projeto simples em JavaScript para calcular o IMC (Índice de Massa Corporal).

O usuário informa nome, altura e peso, e o sistema retorna o valor do IMC e sua classificação.

## Funcionalidades
- Entrada de nome, altura e peso
- Cálculo automático do IMC
- Exibição da classificação (abaixo do peso, normal, sobrepeso, obesidade)
- Validação de campos

## Conceitos praticados
- Manipulação do DOM
- Eventos
- Condicionais (`if / else`)
- Operações matemáticas
- Uso de funções

## Tecnologias
- HTML
- CSS
- JavaScript
